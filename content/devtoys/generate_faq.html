Why publish a FAQ?

A frequently asked questions page on your website will help the visitor find answers quickly thereby improving the overall experience in complex user journeys. It’s also cost efficient. It can save time spent on the phone or answering email.

Answer common questions

It’s a good idea to provide helpful content, not just marketing speak. Answer questions that have been asked by actual users if you can. Otherwise anticipate common questions based on user personas.

Search Engine Optimization FAQ SEO

A well written FAQ is a good first step toward a Search Engine Optimized website. Consider that when people search something, that something is often described in terms of a question. You simply provide the answer to that question.

FAQ Structured Data

Structured data is invisible to human visitors but helps search engine bots understand your site’s FAQ.

Frequently Asked Questions JSON-LD

A rich-text snippet is a semantic structure that, in this case, describes the FAQ questions and answers in a way that the search bot can easily understand. The search engine can then present your answer directly on the SERP (Search Engine Results Page) whenever that question is asked.

JSON Linked Data for SEO

A rich-text snippet is really just a special kind of JSON. So, it looks just like JavaScript, but it is static. It is meant to be read, not executed.

{{< highlight javascript >}} // This snippet when executed in the browser will log a valid JSON-LD FAQ rich snippet // derived from the existing HTML FAQ. // www.theScottKrause.com const jOut = {"@context": "https://schema.org", "@type": "FAQPage","mainEntity": []} const aelQs = [ ... document.querySelectorAll(".l-faq") ] aelQs.forEach(function( elFAQ ){ let elQ = elFAQ.querySelector("summary") let elA = elFAQ.querySelector("p") let oOut = {"@type": "Question", "name": "##Q##", "acceptedAnswer": {"@type": "Answer", "text": "##A##" }} oOut.name = elQ.textContent oOut.acceptedAnswer.text = elA.textContent jOut.mainEntity.push( oOut ) }) console.warn( JSON.stringify( jOut ) ) {{< /highlight >}}
