Frequently Asked Interview Questions


Which languages does WebAssembly support?

.Net, AssemblyScript, C, C#, C++, Clean, Cyber, COBOL, F#, Forth, Go, Grain, Lobster, Lua, Motoko, Nelua, Never, Python, Rust, Rego, TypeScript, and Zig

Can a WASM program access elements on the web page?

WebAssembly cannot directly access the DOM. It calls JavaScript, passing in integer and floating point primitive data types. Thus, to access any Web API, WebAssembly needs to call out to JavaScript, which then makes the Web API call.

Can I author WASM in JavaScript or TypeScript?

Yes, AssemblyScript is like TypeScript (strict variant). Allowing web developers to keep using TypeScript-compatible tooling that they are already comfortable with.
